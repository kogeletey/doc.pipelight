# What is pipelight ?

::: tip tl;dr

It's a tiny executable that execute a list of tasks **you** provided.

:::

Those tasks are commands ordered in steps.
It aims to do the same things as most known CI/CD softwares like Drone.io, Jenkins, GithubCICD.
